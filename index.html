Projects

EntertainMeme

EntertainMeme was my Mobile Applications Project for Android. It started when me and my friend noticed that we kept getting the same memes over and over again. We decided that for our project we should make an application that would show us fresh memes. We decided that the best way to keep on getting the freshest memes is to use the same concept Tinder uses for its applicatin. A swipe feature that will show you the meme, if you like it, you can swipe right and if you do not like it, you can swipe left. We used Android Studio, SQLite and multiple APIs to develop the mobile application.
